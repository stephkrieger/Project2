#Analysis of 2012 New York City High School Students SAT Scores
SDS 192: Project #2
Contributers: Steph Krieger, Ysabella Leonelli, and Eliel Toukam 

This repository includes the code and data used in our analysis of [NYC Open Data](https://opendata.cityofnewyork.us/). The research question we aimed to answer was: What New York City High Schools have the highest SAT score and what qualities do those schools have (e.g. race demographics, quality of teaching, etc.)? Items in this repository are: csv of tidied data set, qmd of our exploratory data analysis, main analysis, and code used to tidy the data, and a pdf of our written report.

Our exploratory data analysis analyzed these variables:
- 'SAT Math Score' = Average SAT math scores for each NYC High School
- 'SAT Reading Score' = Average SAT reading scores for each NYC High School
- 'SAT Critical Writing Score' = Average critical writing scores for each NYC High School
